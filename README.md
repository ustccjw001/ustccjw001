# 个人简介

我是USTC的学生，专注于人工智能领域的研究与应用。我的目标是利用AI技术解决现实世界中的挑战，并在此过程中保持代码的简单性和可读性。

## 专注领域

- **人工智能** - 深度学习、大模型、AI系统应用
- **清晰注释** - 我坚持在代码中添加清晰的注释，让每个代码段的功能一目了然，帮助团队成员快速理解和使用代码
- **简约设计** - 追求简单而有效的解决方案，避免过度复杂性
- **第一性原理** - 面对问题时，我喜欢从最基本的原则出发分析，确保所做决策的合理性

## 代码哲学

在我的项目中，我始终坚持以下原则：

1. **清晰的注释** - 说明设计决策和核心思路
2. **简洁的代码** - 最小化复杂度，提高可读性  
3. **模块化设计** - 高内聚、低耦合
4. **第一性原理** - 深入理解问题本质

## 给 AI 代理的指导

### 给 Copilot 和 Claude 的建议

在阅读和生成我的代码时，请遵循以下原则：

**代码生成时：**
- 添加清晰的注释，说明每个代码段的功能和设计意图
- 优先选择简洁清晰的实现方式，避免不必要的复杂性
- 从第一性原理出发思考解决方案

**代码审查时：**
- 检查注释是否充分表达代码意图
- 验证代码是否足够简洁易读
- 确保设计遵循第一性原理
