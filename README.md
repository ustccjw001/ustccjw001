# 人工智能

在这个仓库中，我们专注于以简单和清晰的代码实现人工智能。我们相信，从基础原则出发，运用简洁的代码和清晰的注释，可以大大提高软件的可维护性和可理解性。

## 代码的简单性
当编写程序时，我们应努力保持代码简洁明了。复杂的代码不仅难以理解，还容易引入错误。坚持使用简单的表达方式和模块化设计，可以使我们的代码既易于阅读又易于维护。

## 清晰的注释
良好的注释可以帮助他人（包括将来的自己）理解代码的目的和逻辑。代码中的每个模块、关键函数和复杂逻辑的意图都应该通过注释加以说明。这种做法有助于减少误解并加快开发速度。

## 第一原则思维
通过从第一原则出发，我们可以深入理解问题，找到更有效和创新的解决方案。避免依赖现有的方法或观点，鼓励思考基础概念，有助于我们在提出新的想法和实现时做到踏实可靠。

## AI 代理的使用说明
### Copilot
- **简洁的代码**: 使用Copilot时，请确保代码尽量简洁。避免复杂的逻辑，利用辅助功能生成简单有效的代码。
- **注释清晰**: 在请求生成代码时，确保提供上下文信息。Copilot会生成更具准确性的代码。

### Claude
- **基础思维**: 在使用Claude时，强调基础原则思维是关键。请清晰定义问题，确保Claude了解从基本概念出发的重要性。

我们鼓励所有贡献者遵循以上原则，共同构建高质量的人工智能代码。
